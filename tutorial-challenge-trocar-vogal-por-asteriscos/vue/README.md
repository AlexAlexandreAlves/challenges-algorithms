# Desafio trocar vogal por asteriscos

**Vamos ao enunciado:**

Usuário digita uma palavra ou frase, e programa retorna a mesma com as vogais trocadas por \*(asterisco).

**Exemplo de Entrada**

```
  a lenda de zelda

```

**Exemplo de Saida**

```
  * l*nd* d* z*ld*

```
